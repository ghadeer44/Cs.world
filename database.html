Database

Database it is a structured group of data that held in a computer and it is an organized set of information, typically stored electronically in a computer system. A database is usually controlled by a database management system (DBMS). Both of them together are referred to as a database system, often shortened to just database.

Data is often designed within the most common types of databases that running today in rows and columns in a series of tables to make processing and querying data efficient. The data can then be accessed, managed, modified, updated, controlled and organized easily. Most databases use SQL to write and query data.

Types of Databases

Object-oriented database

The information in this type of data is represnted in the form of objects, as in object-oriented programming.

Data warehouses

This type of data designed for analysis and fast query, also it is a central repository for data.
Relational databases
This type of data provide the most flexibla and efficient way to access structured information. The items in this data are organized as a set of tables with columns and rows. Relational database became popular in the 1980s.
Distributed database
This type of data is contain two or more files that located in different sites. Also the database may be sorted on multiple computers located in the same physical location or sparse over different networks. .
